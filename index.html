隱私政策

最後更新日期：2024年12月19日

📱 關於本應用程式

「喵喵幸運號碼機」是一個免費的幸運號碼抽選應用程式，讓您輕鬆獲得幸運數字。我們重視您的隱私權，本政策說明我們如何收集、使用和保護您的資訊。

🔍 我們收集的資訊

本應用程式收集的資訊非常有限：

  • 應用程式使用數據：透過 Firebase Analytics 收集匿名使用統計，用於改善應用程式體驗
  • 廣告識別碼：用於顯示個人化廣告，您可以隨時在設備設定中停用
  • 設備資訊：設備型號、作業系統版本等技術資訊，用於應用程式優化

🚫 我們不收集的資訊

  • 個人身份資訊（姓名、電話、地址等）
  • 聯絡人資訊
  • 位置資訊
  • 相機或麥克風存取
  • 其他敏感個人資料

📊 第三方服務

本應用程式使用以下第三方服務：

Firebase Analytics

  • 用途：分析應用程式使用情況，改善用戶體驗
  • 數據類型：匿名使用統計、應用程式崩潰報告
  • 隱私政策：Google 隱私政策

Google AdMob

  • 用途：顯示廣告以維持應用程式免費
  • 數據類型：廣告識別碼、廣告互動數據
  • 隱私政策：Google 隱私政策
  • 廣告設定：您可以在設備設定中停用個人化廣告

🔒 數據安全

我們採取適當的安全措施保護您的資訊：

  • 所有數據傳輸都使用加密技術
  • 我們不儲存個人身份資訊
  • 匿名數據僅用於應用程式改善
  • 定期更新安全措施

👶 兒童隱私

本應用程式適合所有年齡層使用，包括兒童。我們不會故意收集13歲以下兒童的個人資訊。如果發現收集了兒童的個人資訊，我們會立即刪除。

🔄 數據保留

我們只保留必要的匿名數據，用於：

  • 應用程式功能改善
  • 錯誤修復
  • 性能優化

這些數據會在不再需要時自動刪除。

📱 您的權利

您有權：

  • 查看我們收集的數據
  • 要求刪除您的數據
  • 停用廣告追蹤
  • 選擇不參與分析數據收集

⚙️ 廣告追蹤控制

如何停用廣告追蹤：

  • iOS：設定 → 隱私權與安全性 → 追蹤 → 關閉「允許 App 要求追蹤」
  • Android：設定 → Google → 廣告 → 停用「廣告 ID」

🔄 政策更新

我們可能會不時更新本隱私政策。重大變更會透過應用程式更新通知您。繼續使用應用程式即表示您接受更新後的政策。
